# TON Business Project Interaction Guide

## Project Overview

This project is an enterprise-grade Web3 infrastructure solution based on the TON blockchain, specifically designed for the Dubai market. The project contains three core modules: blockchain loyalty engine, zero-code TON payment mini app builder, and frictionless tipping platform.

**Project Type**: Non-code project - Primarily contains business requirements documentation and technical architecture design
**Current Phase**: Requirements analysis and architecture design phase
**Target Market**: Small and medium enterprises in Dubai and Middle East region

## Project Features

### Technology Stack
- **Blockchain**: TON Mainnet/Testnet
- **Frontend Carrier**: Telegram Mini App
- **Smart Contract Language**: FunC
- **Integrated Wallet**: Tonkeeper
- **Backend Services**: Node.js + TypeScript
- **Data Storage**: PostgreSQL + MongoDB + Redis

### Business Value Proposition
1. **Zero Threshold Access**: Complete deployment in 5 minutes, no technical background required
2. **Cross-scenario Interoperability**: Points and payments can be used across different merchants
3. **Deep Ecosystem Integration**: Zero-cost customer acquisition based on Telegram's 1 billion user pool
4. **Ultra-low Transaction Costs**: Leveraging TON blockchain's low-fee characteristics

### Target User Groups
1. **Medium & Large Enterprises**: Shopping malls, airlines, hotel groups
2. **Small & Medium Merchants**: Cafes, restaurants, convenience stores, retail shops
3. **Service Practitioners**: Waiters, hairdressers, tour guides, and other groups needing tip payments
